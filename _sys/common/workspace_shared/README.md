# Engram Common Workspace Resources
This directory stores common resources that are shared across ALL workspaces.

- **Purpose**: "Workspace범위 외에서도 공통으로 사용할 필요" 가 있는 경우의 내용 모아놓을 수 있는 공간.
- **Usage**: Shared scripts, global prompt templates, universal UI assets, or cross-project configuration files should be placed here. They can be referenced securely by any active workspace without duplicating data.
