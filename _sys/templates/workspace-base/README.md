# Engram Workspace Template Base
This directory contains the foundational files and configurations used when initializing a completely new workspace.

- **Purpose**: "Workspace를 처음 구성" 하는 경우의 Base Template.
- **Usage**: When a new project is created, the contents of this directory are copied into the new workspace root. It ensures a standardized, MECE-compliant starting point for any project developed within Engram.
