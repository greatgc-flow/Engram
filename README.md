# Engram

[![Platform: Windows](https://img.shields.io/badge/platform-Windows-0078d4.svg)](https://www.microsoft.com/windows)
[![Python: 3.14](https://img.shields.io/badge/python-3.14-blue.svg)](_sys/runtimes.json)
[![Tests: 808 passing](https://img.shields.io/badge/tests-808%20passing-brightgreen.svg)](_sys/tests/unit)
[![Protocol: 4.2](https://img.shields.io/badge/protocol-4.2-purple.svg)](_sys/ai/protocol.json)

Engram is a portable Windows development environment with a multi-peer AI collaboration system. It provides isolated runtimes, CLI launchers, peer orchestration, automatic model-profile routing, health checks, consensus, and test-driven validation from one relocatable directory.

## Getting Started

1. **Quick Start**: Run `INSTALL.bat` to install dependencies, then `register.bat` to map the drive.
2. **User Manual**: See [`_sys/docs-v2/user/manual.md`](_sys/docs-v2/user/manual.md) for daily workflows, commands, and topology.
3. **Architecture**: See [`_sys/docs-v2/20-architecture.md`](_sys/docs-v2/20-architecture.md) for the layout and system connectivity map.
4. **Agent/Peer Info**: See [`_sys/docs-v2/MOC.md`](_sys/docs-v2/MOC.md) for the Master Index of all peer rules, invariants, and operations.

> **Note to AI Peers**: Do not read this file for operational rules. Please proceed immediately to `_sys/docs-v2/MOC.md` and `_sys/docs-v2/10-invariants.md`.
